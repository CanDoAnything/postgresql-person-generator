# postgresql-training

This repo is a demo of using postgreSQL with both powershell and maybe python.

## Prerequisites

### Required

- Connection to a PostgreSQL database
- Basic understanding of SQL

### Recommended

- VSCode
  - Plugins

    - Markdown Preview Enhanced (Yiyi Wang)
